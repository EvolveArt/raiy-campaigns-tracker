# Raisy Funding - RaisyCampaigns Contract 

A tracker implementation of RaisyCampaigns contract

## Project Setup
```
npm install 
``

## Compile
```
npm run start / npm run dev (nodemon)
```

#### .env file sample
```
NETWORK_RPC=
NETWORK_CHAINID=
DB_URL=
API_ENDPOINT=
CONTRACTADDRESS=
DECIMAL_ENDPOINT=
```
